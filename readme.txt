Adds a custom post type "Notification" and displays vertical scrolling notifications via shortcode or auto-injection on selected pages. Ideal for schools, colleges, news, and more.

== Description ==

🎉 **Vertical Notifications** lets you create and manage announcement-style messages using a custom post type and display them on the frontend with a shortcode or automatic injection into a selected page.

Perfect for schools, colleges, businesses, or news portals where updates and notices are essential.

= Features =

- ✅ Custom Post Type: `Notification`
- ✅ Show latest notifications as a vertical ticker
- ✅ Use shortcode `[vertical_notifications]` anywhere
- ✅ Auto-inject notifications on a selected page
- ✅ Open notification links in a new tab (optional)
- ✅ Elementor compatible
- ✅ Admin dashboard widget
- ✅ Scroll speed and limit controls
- ✅ Simple, responsive, and lightweight

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu.
3. Go to **Settings > Vertical Notifications** to configure options.
4. Use shortcode `[vertical_notifications]` where needed.

== Usage ==

Add this shortcode where you'd like to show the notifications:

`[vertical_notifications]`

You can also enable auto-injection to show notifications automatically on a selected page.

== Frequently Asked Questions ==

= How do I display notifications on a page? =  
Use the shortcode `[vertical_notifications]` or enable "Auto Inject" from plugin settings.

= Can I control how many notifications are shown? =  
Yes! You can set the number in plugin settings.

= Can I change scroll speed? =  
Yes, set your preferred scroll speed in plugin settings.

= How do I show the "NEW" label? =  
This feature is part of the Pro version. Mark notifications as "NEW" using a checkbox when editing the post.
